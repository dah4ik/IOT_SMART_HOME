# IOT_SMART_HOME

## Smart Home IoT Monitoring and Control System

IOT_SMART_HOME is a virtual Smart Home IoT project built with Python, MQTT, SQLite, PyQt5 GUI, REST API, rule engine, alarms and analytics.

The system simulates several IoT devices and manages them through a central Data Manager application.

## Main Features

- MQTT communication
- Device emulators
- Data Manager
- SQLite database
- Rule Engine
- PyQt5 desktop GUI
- REST API
- Web dashboard
- Analytics module
- Assistant BOT

## Devices

The project includes the following virtual devices:

1. DHT temperature and humidity sensor
2. Light sensor
3. Electricity and water meter
4. Reed door sensor
5. Button event producer
6. Relay actuator
